Использование пакета laravel-livewire для написание простых запросов Ajax.
Установка :
Выполнить команду : 
composer update;
php artisan migrate ;
php artisan storage:link(при необходимости);
Переименовать файл .env.example в файл .env и настроть подключение к базе данных;

Стартовая страница :
public\index.php
